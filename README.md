# Практичне заняття №8
## Оброблення виняткових ситуацій

У цьому репозиторії розміщено програмний код і звіт до практичного заняття №8, присвяченого обробленню виняткових ситуацій у мові **C++**.

### Зміст репозиторію
- `ConsoleApplication1.cpp` — вихідний код програми
- `ConsoleApplication1.vcxproj` — файл проєкту Visual Studio
- `ConsoleApplication1.vcxproj.filters` — службовий файл Visual Studio
- `ConsoleApplication1.vcxproj.user` — користувацькі параметри Visual Studio
- `Практична 8.pdf` — готовий звіт до роботи

### Короткий опис роботи
У роботі досліджується програма класу `Vect`, у якій опрацьовано:
- причину виникнення помилки під час використання поля типу `char` для зберігання розміру масиву;
- виправлення помилки без зміни типу `char`;
- перевірку меж індексу в перевантаженому операторі `[]`;
- використання блоків `try/catch` для перехоплення винятків;
- реалізацію безпечного деструктора.
